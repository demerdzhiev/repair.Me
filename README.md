# repair.Me
## SoftUni React exam project. This repository is for educational purposes only!

> What’s This About?
> ```
> * "repair ME" is a directory for home renovation services. You can browse through different services, check out their details, and manage
> * them if you're a service provider. It’s designed to be user-friendly for both clients looking for renovation help and providers 
> * offering their services.
```

> The application allows you to:
> ```
> * Browse through a catalogue of home renovation services
> * View detailed information about each service
> * Add comments to services
> * Create, edit, or delete services if you're a service provider
> ```


## Usage
> **Browse Services**
>   - Navigate through the catalogue to explore different home renovation services.
> 
> **Service Details**
>   - Click on a service to view detailed information and add comments.
> 
> **Manage Services**
>   - Service provider users can create, edit, or delete services.

## How to Get Started
> To run the application, you need to execute the following commands:
> ```
> * In the server folder, run: `node server.js`
> * In the client folder, run: `npm run dev`
> ```
> No additional setup is required beyond these commands.

## Default Setup
> On the first run of the application, you can use the default server credentials:
> ```
> * Username: admin@abv.bg
> * Password: admin
> ```
> A default service is seeded in the server for initial setup.
